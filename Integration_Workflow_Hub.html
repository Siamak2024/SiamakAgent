Arbetsflöde & Status

Hur fungerar Integration Workflow?

📋 Översikt

EA 2.0 Integration Workflow guidar dig genom en strukturerad process för att bygga en komplett Enterprise Architecture-modell, från affärsstrategi till teknisk implementation.

🔄 Arbetsflödet (4 faser)

  1. Business Model Canvas: Definiera affärsmodellen
  2. Value Chain Analysis: Analysera värdekedjan och identifiera AI-möjligheter
  3. Capability Mapping: Kartlägg capabilities per domän
  4. Strategy Workbench: Skapa Wardley Map för strategisk positionering

✅ Så fungerar det

  • Varje fas måste slutföras innan nästa låses upp
  • Efter varje fas, klicka "Exportera (EA)" i verktyget
  • Datan exporteras automatiskt till integration cache
  • Nästa fas låses upp när export är klar
  • Efter alla 4 faser → Importera i EA Platform

💡 Tips

  • Använd "Ladda exempel"-knappen för snabb demo
  • Klicka "Spara" eller "AI Analysera" för auto-export
  • Grön bock = Fas slutförd
  • Blå pulsering = Aktuell fas
  • Lås = Fas ännu ej tillgänglig

🎯 Vad händer i EA Platform?

När alla faser är slutförda kan du importera datan till EA Platform:

  • Capabilities från Value Chain + Capability Map
  • Systems från Value Chain digitalisering
  • AI Agents från Value Chain AI-opportunities
  • Wardley-positionering från Strategy Workbench
  • Build/Buy/Partner rekommendationer
